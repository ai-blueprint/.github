### 💡 核心理念
我们正在构建一个**全新的编辑器**。
传统的 AI 架构设计与测试是冰冷的代码，而“炼丹蓝图”将其转化为**直观的、逻辑清晰的蓝图**来轻松设计和测试。

- **可视化建模**：拖拽节点，构建你心目中完美的新架构。
- **底层洞察**：不只是画图，每一个节点都直观对应 PyTorch 的组件，既让模型结构一目了然，又能帮助你快速调试。
- **社区共建**：你可以设计自己的架构并一键分享给全球开发者。
